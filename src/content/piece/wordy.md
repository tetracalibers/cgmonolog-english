meaning: 文章などが「とても長い」

examples:
  ########################################
  - en: This comment’s a bit too wordy. Might rewrite it with just the essentials.
    ja: このコメント、ちょっと冗長だな…。必要なことだけに書き直そうかな。
    scene: 自分のコードコメントを見直しているとき
  ########################################
  - en: I know I get wordy in docs… it’s just hard to explain these edge cases simply.
    ja: ドキュメント、つい冗長になっちゃうんだよね…。こういうイレギュラーケース、簡潔に説明するの難しくてさ。
    scene: ドキュメント作成に苦戦しつつ少し照れながら
  ########################################
  - en: The UI error messages used to be super wordy—we trimmed them down last sprint.
    ja: 昔はUIのエラーメッセージ、すごくまわりくどかったんだよね。先週のスプリントで簡潔にしたよ。
    scene: 過去の改善を振り返っているとき
  ########################################

「言葉数が多すぎる」「冗長な」「まわりくどい」といった意味の形容詞

- 文章や説明がくどくて長い、または簡潔さに欠ける場合によく使われる
